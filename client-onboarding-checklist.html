Client Onboarding Checklist

From Pixel Install to Campaign Launch

Overall Progress 0 / 0
1
Discovery & Setup
Week 1
🔍 Client Intelligence
  • Identify client's target market: B2B or B2C
  • Define ICP(s) — for B2B: company firmographic (revenue, headcount, industry)
  • Define Avatar(s) — for B2B: Buyer, Champion, End User roles within ICP
  • If B2B: map awareness stages per avatar (Problem / Solution / Product Aware)
  • Research client's competitors (OrchestraOS agent-assisted)
  • Audit client's existing digital assets (website, social, ads)
  • Scrape client's social media for repurposable assets
  • Define client's offer / value proposition per avatar
  • Determine channels: Meta, DSP, cold email, LinkedIn (based on B2B/B2C and budget)
📡 ArkData Pixel Installation
  • Get access to client's website backend (CMS admin or developer contact)
  • Install ArkData pixel (v4 for accuracy, v3 for richer data — evaluate per client)
  • Set up white-labeled dashboard with client branding
  • Verify pixel is firing correctly (check visitor/company data flowing)
  • Walk client through dashboard — instant dopamine moment
  • Set up click tracking and session recording
🎯 ListMagic Audience Building
  • Build intent keyword lists per avatar and awareness stage
  • Create 3 target markets x 3 intent topics = 9 audiences (adjust per client)
  • Generate interactive map of target prospects
  • Validate audience quality (size, relevance, freshness)
  • Present audiences to client for approval
2
Creative Production
Week 1 – 2
🎨 Brand Setup
  • Collect client brand guide (colors, fonts, logos, tone of voice)
  • Define copy messaging per avatar per awareness stage
  • Create 3–5 copy variations per avatar
🖼 Static Display Ads (DSP)
  • Tier 1: 3–5 variations x 3 primary sizes (728x90, 300x250, 160x600)
  • Tier 2: 3 variations x 3 secondary sizes (336x280, 300x600, 320x50)
  • Per avatar — multiply above by number of avatars
  • Client approval on all creatives before launch
🎬 Video Assets (Meta)
  • 1–3 video variations per avatar (reels, stories, square)
  • Hooks: test 3–5 different opening hooks per video concept
  • Ensure brand consistency across all formats
📄 Landing Pages
  • 1 landing page per avatar (minimum)
  • A/B test versions where budget allows
  • Mobile-optimized
  • Tracking pixels installed on all landing pages
  • Thank-you page / confirmation flow set up
3
Campaign Setup
Week 2
📊 DSP Campaigns
  • Upload all display creatives to DSP platform
  • Configure targeting with ListMagic intent audiences
  • Set frequency capping (prevent overexposure)
  • Configure brand safety: allowlists / blocklists
  • Layer verification tools (IAS / DoubleVerify if available)
  • Set bid strategy and daily budget
  • Set up per-avatar ad sets
📱 Meta Campaigns
  • Upload video and image assets
  • Create custom audiences from ListMagic data
  • Set up CBO for single-offer campaigns
  • Configure A/B tests (copy, creative, audience)
  • Install Meta pixel on landing pages for conversion tracking
✉ Email Campaigns (if B2B national)
  • Prepare email copy (7–9 email drip sequence per avatar)
  • Hyperpersonalize each email
  • Configure sending infrastructure (warm domains, DMARC/SPF/DKIM)
  • Validate deliverability before full launch
🔗 LinkedIn (if B2B high-value)
  • Identify high-intent prospects from ListMagic data
  • Prepare targeted InMail or connection request copy
  • Only for prospects who've shown multiple intent signals
4
Launch & Monitor
Week 2 – 3
🚀 Simultaneous Launch
  • All channels go live at the same time (DSP + Meta + Email)
  • Verify all tracking is firing correctly
  • Monitor first 24–48 hours for anomalies
💻 Client Dashboard Setup
  • Configure funnel view: audience → impressions → landing page → conversion
  • Attribution tracking across all channels
  • Per-campaign and per-avatar filter views
  • Set up retargeting: landing page visitors enter new custom audience
🔄 Ongoing Management
  • Weekly performance review
  • Kill underperforming ad variations within first week
  • Creative refresh every 2–4 weeks
  • Report to client: metrics, insights, next steps
  • Identify cross-sell opportunities (pixel → intent data → creative)
⚠ Blockers to Watch For
Client doesn't have CMS access for pixel install → need their developer
Client's CRM isn't API-integrated → need integration call
Client has only one avatar → simpler campaign but less testing surface
Client is local only → intent data works best national; adjust expectations
